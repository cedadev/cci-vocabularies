This concept scheme represents the data types used by the CCI project
The main data type in a data set used by the ESA Climate Change Initiative (CCI).
<p>The CCI concept schemes are used to define concepts within an element of a Directory Reference Scheme. There are currently eight concept schemes defined: <a href=../dataType>data type</a>, <a href=..//ecv>ecv</a>, <a href=../scheme/cci/freq>frequency</a>, <a href=../org>institutes</a>, <a href=../platform>platforms</a>, <a href=../platformGrp>platform groups</a>, <a href=../platformProg>platform programs</a>, <a href=../procLev>processing levels</a>, <a href=../product>products</a>, <a href=../project>projects</a>, and <a href=../sensor>sensors</a>. The use of a concept scheme allows easy navigation up and down the tree.</p><h5>SKOS Navigation Down the Concept Scheme</h5><img src=conceptSchemeDown.svg /><h5>SKOS Navigation Up the Concept Scheme</h5><img src=conceptSchemeUp.svg /><p>As well as being concepts the the concepts are also named individuals of the <a href=../../../ontology/cci>CCI Ontology</a> classes.<h5>OWL Classes</h5><img src=classes.svg /><p></p>
